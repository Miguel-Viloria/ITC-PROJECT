Course Subjects

HTML or Hypertext Markup Language is the backbone of every webpage, providing the structure and layout of content. In the Introduction to Computing course, students learn how to use HTML to organize text, images, links, and other multimedia elements on a page. By mastering tags and attributes, students gain the skills to create clear, well-structured content that forms the foundation of any website.

CSS or Cascading Style Sheets allows students to style and enhance their HTML content, transforming basic web pages into visually appealing designs. In this course, students learn how to use CSS to control layout, color, fonts, and spacing, creating a polished look that aligns with modern design principles. With CSS, students can customize the visual identity of a webpage, making it more engaging and user-friendly.

JavaScript brings interactivity and dynamic functionality to websites, enabling students to make their webpages more responsive and engaging. In the Introduction to Computing course, students explore how JavaScript can be used to create interactive elements like buttons, forms, and animations. By learning basic JavaScript programming, students acquire the tools to make their websites more interactive, adding functionality that enhances the user experience.

The foundational ideas of computers and information technology are introduced in this course. Students will obtain essential computing skills and an awareness of how technology affects all facets of life through theoretical talks, practical laboratory exercises, and the application of software tools. Learners will also gain technical skills in creating websites using basic HTML, CSS and JavaScript.


Course Outcomes

1. Describe the components and functions of a computer system
2. Differentiate between various types of computers and mobile devices.
3. Identify various types of software applications and their uses.
4. Utilize word processing software for document creation and formatting.
5. Apply spreadsheet software for data analysis and basic formulas.
6. Create presentations using presentation software effectively.
7. Understand the concept of number systems and their importance in computing.
8. Navigate the internet and use search engines efficiently.
9. Discuss ethical considerations and societal impact of technology.
10. Create a website using Basic HTML, CSS and JavaScript.

The course covers the use of general-purpose programming language to solve problems. The emphasis is to train students to design, implement, test, and debug programs intended to solve computing problems using fundamental programming constructs.

Program Objectives

1. Identify and explain the different programming terms.
2. Explain the steps in the Program development life cycle/steps in problem-solving and program development.
3. Apply flowcharting in designing program solutions.
4. Design. implement, test. and debug a program, based on a course specification, that uses the following fundamental programming components: (a) primitive data types, (b) basic computation, (c) simple I/0, (d) conditional and iterative structures, (e) definition of functions and parameter passing, and (f) recursion.
5. Assess and recommend revisions to another programmer's code (a) regarding documentation and program style standards that contribute to readability and maintainability of software, (b) regarding the appropriateness of chosen conditional and iterative constructs given a programming task, and (c) regarding thoroughness in applying procedural abstraction.
